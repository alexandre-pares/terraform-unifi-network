# Wifi Broadcast Terraform Module

This module manage a single wifi broadcast managed by your Unifi Network application.

## Examples

- [Create Wifi Broadcast](../../examples/create_wifi_broadcast)

## Limitations

- `PASSPOINT` hotspot configuration is not supported by the API endpoint (but can be returned by the `network_details` module).

Do not create a wifi broadcast if you are wireless connected to the same upstream device and using a local endpoint: you will likely lose connection.
Instead, use a wired connection or use a different upstream device.
