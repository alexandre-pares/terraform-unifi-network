# Network Terraform Module

This module manage a single network managed by your Unifi Network application.

## Examples

- [Create network](../../examples/create_networks)

## Limitations

Only network type `UNMANAGED` and `GATEWAY` are tested. `SWITCH` types should work but aren't tested (I don't have the hardware to test - required L3 managed switch).
